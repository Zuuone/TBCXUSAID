
## Used Technologies

1. **HTML**: For structuring the content of the web pages.
2. **CSS**: For styling the HTML elements and providing a responsive layout.
3. **JavaScript**: For adding interactive behavior to the web pages.
4. **SVG**: For scalable vector graphics, such as logos and icons.
5. **Media Queries**: For responsive design to ensure the website looks good on different devices.

## Project Overview

### Sections

1. **Header**:

2. **Hero Section**:

- Showcases different product offerings such as "ციფრული," "პრემიუმი," and "360."

4. **Awards Section**:

### Key Features

- **Responsive Design**: The website is designed to be responsive, ensuring it looks great on devices of all sizes.
- **Interactive Elements**: Uses JavaScript for interactive elements such as scrolling and hover effects.
- **SVG Icons**: Utilizes SVGs for crisp and scalable icons and logos.
